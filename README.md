# Global Climate Change Analysis

## 📌 Project Overview
This repository contains a data analysis workflow that explores **global climate change trends** using publicly available datasets. The objective is to understand historical changes in key climate indicators (e.g., temperature, CO₂ emissions, sea level) and visualize these trends in Tableau.

## 🗂️ Repository Structure
- `Raw Data/` – Original downloaded datasets (source: Kaggle).
- `Clean Data/` – Processed datasets after cleaning and transformation using Power Query Editor.
- `Result/` – Tableau workbook (`Global Climate Change.twb`) containing visualizations and dashboards.
- `Description/Description.txt` – Additional notes about the dataset and analysis.

## 📥 Data Source
- **Source:** Kaggle
- **Description:** Contains global climate data (temperature, emissions, sea level, etc.) collected from reputable scientific sources.

## 🧹 Data Cleaning (Power Query Editor)
Data preparation was done using **Power Query Editor** (part of Excel) and includes:
- Removing duplicate or irrelevant rows
- Standardizing column names and data types
- Filtering out invalid/missing values
- Creating calculated fields needed for analysis
